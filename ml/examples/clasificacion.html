Clasificando figuras para apurados

Este clasificador clasifica tres figuras, triangulos, circulos y cuadrados. Fue entrenado con la libreria ml5.js, la cual esta hecha encima de la libreria de tensorflow.js, la cual se utiliza para modelar redes neuronales.

El primer paso que tome, fue generar un dataset de 900 figuras, 300 imagenes de circulos, 300 imagenes de cuadrados y 300 imagenes de triangulos, con la ayuda del ambiente de programacion en java Procecssing. Si gustas ver el dataset especifico con el cual se entreno el modelo, lo puedes descargar de aqui.

El tipo de red neuronal que utilizamos, es una red neuronal convolucional, puedes ver la documentacion del codigo de entrenamiento aqui.
Desafortunadamente, como el modelo es entrenado localmente, pero mediante el navegador, solo pudimos cargar 600 imagenes en lugar de 900, pero aun asi logre que entrenara con 50 epochs. ml5 contiene una funcion que te permite ver en vivo como la funcion de perdida va desarrollandose conforme pasa el tiempo.

La funcion de perdida del modelo se ve de la siguiente manera.

Despues del entrenamiento, exportamos el modelo, junto con toda la informacion de ajustes de pesos, ml5 se encarga de guardar esto en tres archivos, los cuales podemos encontrar en este repositorio de github.
Por ultimo cargamos el modelo a un proyecto regular de p5 junto con la libreria de ml5.js y programamos para tomar muestras de la camara web y hacer las predicciones en tiempo real! Puedes ver la documentacion de esta parte aqui.
